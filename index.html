Css Selectors and what they do

There are about 6 CSS selector and this is what they do:

Selector Example Selects
* * It's a global selector. It selects all elements
#id #email The element with id="email"
.class .classname All elements with class="classname"
element p Selects all <h1> elements
[attribute] [target] Selects all elements with a target attribute
Pseudo-class Selector like :active a:active All active links
